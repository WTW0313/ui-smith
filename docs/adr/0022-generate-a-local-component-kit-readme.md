# Generate a Local Component Kit README

UI Smith v1 will generate a local Kit README near the Component Kit alongside the root Kit Manifest. The Kit Manifest gives Coding Agents structured data, while the Kit README gives humans and agents readable usage rules, customization guidance, verification commands, and component examples.

## Consequences

The Kit README should not replace the Kit Manifest or duplicate every schema field. It should explain how to use and extend the Component Kit in the host repository's language and should be updated when UI Smith changes generated components.
